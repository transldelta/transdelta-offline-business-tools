# Start here

Upload these files to your new GitHub repository:

https://github.com/transldelta/transdelta-offline-business-tools

Files:

- index.html
- README.md
- llms.txt
- robots.txt
- sitemap.xml
- LAUNCH_POSTS.md
- FREE_DISTRIBUTION_MAP.csv

Then activate GitHub Pages:

Settings → Pages → Deploy from a branch → main → /root → Save

Your expected live URL:

https://transldelta.github.io/transdelta-offline-business-tools/

Wait 1 to 5 minutes after saving GitHub Pages.
